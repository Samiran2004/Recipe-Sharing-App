## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`DATABASE_URI`

`CLOUD_NAME`

`CLOUD_API_KEY`

`CLOUD_API_SECRET`

`JWT_SECRET_KEY`

`NODEMAILER_USERNAME`

`NODEMAILER_PASSWORD`
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

```bash
  cd my-project
```

```bash

```

```bash
  npm run start
```
